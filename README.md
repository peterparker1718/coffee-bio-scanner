# Coffee Bio Scanner

**AI-Powered Coffee Bean Quality Analyzer · FDA-Inspired Biosecurity Pattern · Gemini 1.5 Flash + FastAPI**

A biosecurity-style scanning system for specialty coffee supply chains. Uses Gemini 1.5 Flash multimodal analysis on bean imagery to surface quality, contamination, and origin-verification signals at import scale.

Built for [Java Bridge Coffee](https://github.com/peterparker1718/java-bridge-workflow-tool), an Indonesia-to-US specialty importing operation. First-to-market technology in this niche.

## Why this matters

Coffee importers currently rely on manual inspection or send samples to third-party labs. Both are slow and expensive. A vision-model layer can pre-screen lots in minutes instead of days, flagging anomalies before container-level cost accrues.
